Готовые настройки Shadowrocket для на Wi-Fi, LTE, LTE+БС



Для Wi-Fi и LTE без белых списков рекомендуется использовать ru.conf

Apple(кроме FaceTime и iMessage) - напрямую

Домены и CIDR IP из белых списков - напрямую

Айпи-трекеры - напрямую(можно сделать REJECT, для полного обрыва соединения)

Запросы из мессенджера Мах - напрямую(список будет пополняться, собираю вручную)

Все домены ru, su, рф - напрямую

Прочие российские домены - напрямую

TikTok - проксируется(пришлось добавлять отдельно, т.к. из-за предыдущего пункта не работал)

Весь остальной трафик - напрямую



При белых списках используйте ru_whitelist.conf

Apple - весь трафик проксируется(работают уведомления)

Домены и CIDR IP из белых списков - напрямую

Айпи-трекеры - напрямую(при ограничениях соединение обрывается, т.к. этих адресов нет в БС)

Запросы из мессенджера Мах - напрямую(список будет пополняться, собираю вручную)

Весь остальной трафик - напрямую
